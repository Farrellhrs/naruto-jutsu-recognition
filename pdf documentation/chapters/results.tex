\chapter{Results}

\section{Quantitative Summary}
The unified model achieved strong test accuracy and a successful CoreML export for on-device use.

\begin{itemize}[leftmargin=2em]
\item Test accuracy: 96.85\%
\item Feature dimension: 126
\item Usable samples: 3961
\item One-hand coverage: 83.03\%
\item Two-hand coverage: 16.97\%
\end{itemize}

\section{Model Behavior Observations}
\begin{itemize}[leftmargin=2em]
\item Most classes are separated reliably under clean lighting and stable framing.
\item Minor confusions occur among similar finger formations.
\item One-hand fallback through zero padding improves practical robustness.
\end{itemize}

\section{iOS Runtime Observations}
\begin{itemize}[leftmargin=2em]
\item Real-time inference supports responsive sequence gameplay.
\item Combined hand and face cues improve fire-style effect direction logic.
\item Battle mode adds meaningful interaction through defend and attack windows.
\end{itemize}
